# 🎯 Number Guessing Game

A simple number guessing game built using JavaScript.

---

## 🚀 Features

- Random number generation
- User input via prompt
- Hint system (Too high / Too low)
- Win detection

---

## 🛠️ Technologies Used

- HTML
- JavaScript (Vanilla JS)

---

## ▶️ How to Play

1. Enter a maximum number
2. Guess the number
3. Follow hints:
   - Too high
   - Too low
4. Win when correct!

---

## ⚠️ Limitations

- Uses prompt (no UI)
- No score tracking
